Despliegue de Proyecto Angular en GitHub Pages

Aquí te mostramos los pasos detallados para desplegar tu proyecto Angular en GitHub Pages.

1. Configuración Inicial del Proyecto

Asegúrate de que tu proyecto Angular esté correctamente configurado. Primero, verifica que el atributo baseHref esté bien configurado en el archivo angular.json.

En la sección de build, dentro de angular.json, asegúrate de tener algo como esto:

{{ '{' }} "projects": {{ '{' }} "tu-proyecto": {{ '{' }} "architect": {{ '{' }} "build": {{ '{' }} "options": {{ '{' }} "baseHref": "/ProyectoAngular/" {{ '}' }} {{ '}' }} {{ '}' }} {{ '}' }} {{ '}' }} {{ '}' }}

2. Compilación del Proyecto para Producción

Compila el proyecto para producción usando el siguiente comando. Esto generará los archivos listos para el despliegue en la carpeta dist/.

ng build --configuration production

Este comando creará una carpeta dentro de dist/nombre-del-proyecto/browser que contiene todos los archivos necesarios para el despliegue.

3. Despliegue a GitHub Pages

Usa el siguiente comando para desplegar tu proyecto a GitHub Pages. Asegúrate de ejecutar este comando desde la raíz del proyecto. Tiene que aparecer el index.html

npx angular-cli-ghpages --dir=dist/nombre-del-proyecto/browser

Este comando subirá los archivos generados en la carpeta browser a la rama gh-pages de tu repositorio en GitHub.

Si todo va bien, verás un mensaje similar a este:

🌟 Successfully published via angular-cli-ghpages! Have a nice day!

4. Verificar el Despliegue

Una vez completado el despliegue, podrás acceder a tu aplicación en GitHub Pages en la siguiente URL:

https://danielcontreras205.github.io/ProyectoAngular/

Reemplaza danielcontreras205 con tu nombre de usuario de GitHub y ProyectoAngular con el nombre de tu repositorio.

5. Automatizar el Despliegue con un Script

Para hacer el proceso más sencillo, puedes agregar un script en tu archivo package.json para automatizar el despliegue.

{{ '{' }} "scripts": {{ '{' }} "deploy": "ng build --configuration production && npx angular-cli-ghpages --dir=dist/nombre-del-proyecto/browser" {{ '}' }} {{ '}' }}

Luego, puedes ejecutar el siguiente comando para compilar y desplegar en un solo paso:

npm run deploy
